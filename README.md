# VetAPI Veteriner Randevu Takip Sistemi

Bu proje, veteriner randevularını yönetmek ve takip etmek amacıyla geliştirilmiş bir sistemdir.

## Sınıflar
- Customer
- Doctor
- Animal
- Vaccine
- Appointment
- AvailableDate
